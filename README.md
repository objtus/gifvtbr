# gifvtbr

GIF ベースの VTuber アバターエンジン。配信ソフト（OBS 等）と組み合わせて使うデスクトップアプリです。

## 特徴

- **GIF アニメ対応** — ループ再生・単発まばたきをフレーム単位で制御
- **9 レイヤー合成** — 背景・胴体・手（左右独立）・口・目・頭部・前景を Canvas で重ね描き
- **リップシンク** — マイク音量（RMS）に応じて口が自動で開閉
- **まばたき** — 2 枚切替 / 3 枚切替 / GIF 単発の 3 モード
- **手レイヤー自動切替** — マウス操作 → 右手差分、キーボード操作 → 左手差分
- **差分スロット × 12** — 表情・衣装をショートカットキー（Ctrl+1〜9 / Ctrl+↑←→）でリアルタイム切替
- **グローバルホットキー** — OBS にフォーカスがある状態でも差分切替が動作
- **前回状態の自動復元** — 起動するとプロジェクトと設定が自動で復元される

## 動作環境

- Windows 10 / 11（macOS は未検証）
- Node.js 18.x 以上

## セットアップ

```bash
npm install
npm start
```

## OBS との連携

アプリ起動後、OBS の **ウィンドウキャプチャ** でこのアプリのウィンドウを選択してください。  
ウィンドウは透過・フレームなしで動作するため、クロマキー不要で背景を透過できます。

## ショートカット

| キー | 動作 |
|---|---|
| `S` | 設定パネルを開く |
| `Ctrl + 1〜9` | 差分スロット 1〜9 を適用（同キーで解除） |
| `Ctrl + ↑ / ← / →` | 差分スロット 10〜12 を適用 |
| `Ctrl + 0` / `Ctrl + ↓` | ベース状態に戻す |
| `F12` | DevTools を開く |

## プロジェクトの保存・読込

設定パネルの「💾 プロジェクト」タブから JSON 形式で保存・読み込みできます。  
画像データは Base64 で JSON に埋め込まれるため、ファイル 1 つで持ち運び可能です。
