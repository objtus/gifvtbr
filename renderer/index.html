💾 プロジェクト
ベースレイヤーは「通常時」の全パーツ画像です。
差分スロットでは、ここで登録した各レイヤーを任意に上書きして表情・衣装差分を作成します。
🌆 背景 (bg)
🧍 ボディ胴体 (body_torso)
胸から下・衣装胴部。頭部とは別に追従遅延を設定できます。
🗣️ ボディ頭部 (body_head)
頭・髪・首。目・口パーツと同じ遅延で動きます。胴体より少し遅らせると自然です。
🤝 手レイヤー（右手 / 左手 独立）
右手・左手を別々のレイヤーとして重ね合わせます。
マウス差分は右手だけ、キーボード差分は左手だけを変えれば「両方同時」も自動で成立します。

👁️ 目 (eyes)
まばたきタブでモード設定。差分で「困り眉」などに上書き可能です。
👄 口 (mouth)
以下は帯域分割モード ON 時のみ使用される形状素材です（任意）。未登録の場合は音量ベースの口にフォールバックします。
✨ 追加レイヤー (extra / fg)
差分スロットはベースレイヤーへの「パッチ」です。
各スロットで「どのレイヤーをどの画像に差し替えるか」だけを指定します。
指定しないレイヤーはベースのままになります。
例：「困り顔」→ 目レイヤーだけ差し替え。口・ボディはそのまま。
ショートカット：Ctrl+1〜9 / Ctrl+↑←→（同キーで解除）・Ctrl+0 / Ctrl+↓ でリセット
ワンショットはトリガーして終わる「イベント」です。差分スロットとは独立したレーンです。
再生中は登録レイヤーをベース/差分の上に上書きし、終了後は元の状態に自動復元します。
ショートカット：Ctrl+Alt+1〜8（同キーで停止）

🎛️ 帯域分割モード（施策D）
ONにすると全域RMSの代わりに中域（460〜1840Hz）を口開き判定の主軸にします。声の「実体」に反応しやすくなります。

💾 プロジェクト保存
設定・全レイヤー画像・差分スロットをまとめてJSONファイルに書き出します。
画像はBase64でJSON内に埋め込まれるため、ファイル1つで持ち運びできます。
📂 プロジェクト読み込み
以前に保存したJSONを読み込み、設定と画像を復元します。
現在の設定は上書きされます。
🌐 ローカルAPI
localhost の指定ポートで HTTP サーバーを起動します。StreamDeck・コメントビューワー等の外部ツールから差分切替・アクション再生を制御できます。
● 停止中
エンドポイント例:
GET /variant/0 差分スロット0を適用
GET /variant/reset ベースに戻す
GET /action/0 アクション0を再生
GET /action/わらい?loop=3 名前・ループ指定
GET /action/stop アクション停止
GET /status 現在の状態をJSON取得
🌬️ GIFループ揺らぎ設定
GIF画像に微細なランダム揺れを加えて自然感を演出します。0で無効。
