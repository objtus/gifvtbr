💾 プロジェクト
ベースレイヤーは「通常時」の全パーツ画像です。
差分スロットでは、ここで登録した各レイヤーを任意に上書きして表情・衣装差分を作成します。
🌆 背景 (bg)
🧍 ボディ胴体 (body_torso)
胸から下・衣装胴部。頭部とは別に追従遅延を設定できます。
🗣️ ボディ頭部 (body_head)
頭・髪・首。目・口パーツと同じ遅延で動きます。胴体より少し遅らせると自然です。
🤝 手レイヤー（右手 / 左手 独立）
右手・左手を別々のレイヤーとして重ね合わせます。
マウス差分は右手だけ、キーボード差分は左手だけを変えれば「両方同時」も自動で成立します。

マウス操作 差分
左手 (hand_l)
キーボード操作 差分
右手_マウス ＋ 左手_キーボード = 両方同時 ✓

👁️ 目 (eyes)
まばたきタブでモード設定。差分で「困り眉」などに上書き可能です。
👄 口 (mouth)
✨ 追加レイヤー (extra / fg)
差分スロットはベースレイヤーへの「パッチ」です。
各スロットで「どのレイヤーをどの画像に差し替えるか」だけを指定します。
指定しないレイヤーはベースのままになります。
例：「困り顔」→ 目レイヤーだけ差し替え。口・ボディはそのまま。
ショートカット：Ctrl+1〜9 / Ctrl+↑←→（同キーで解除）・Ctrl+0 / Ctrl+↓ でリセット
